4. Testning

Jag testade min hemsida i Chrome genom live server, resultatet var jag jättenöjd med för att det såg ut som jag ville att det skulle se ut. Det funkade även väldigt bra, alla funktioner fungerade som det skulle och jag var väldigt nöjd med den. Firefox hade samma resultat, men däremot stötte jag på ett problem när det kom till mobilen. Det visa sig att hemsidan ser ganska bra ut, den uppfyller samma kvalite som i Chrome men, storleken på texten var lite för stor i mobilform. Därför anpassade jag textstorleken i @media raderna för att det man skulle se vad som stod och att det passade fint, inte att allt text går utanför fyrkanten. Det uppstod samma problem runt serachbaren, och jag behövde ändra storlekar för att det skulle passa in. När jag minskade ner hemsidan såg allt väldigt bra ut, trodde att det skulle vara likadant för mobilen men det visade sig att texten där var för stor vilket gjorde att bilderna inte syntes alls men däremot såg resten ganska bra ut och jag känner mig nöjd.

5. Utvärdering

Jag tycker att min planering fungerade ganska bra under arbetets gång, jag kände att den hjälpte mig påminna om vad jag gjorde förra veckan, samt vad som behövdes göras. Planeringen var även bra för att det underlättande, jag behövde inte stressa för att jag visste att ifall jag gör dessa saker denna dagen behöver jag inte oroa mig för de kommande veckorna då jag har gjort tillräckligt. Det som gick mindre var att jag inte skrev direkt efter lektionerna, jag borde ha skrivit direkt efter alla arbetens gång vilket skulle göra allt mycket enklare. Dessutom skulle det vara mer detaljerat skrivet än detta för att jag inte minns så mycket, ifall jag hade skrivit efter lektionen hade kunnat skriva mycket för att jag visste vad jag gjorde på de dagarna. Jag tyckte att det utseendemässigt ser väldigt bra ut, det är en fungerande hemsida med fina färger, fonter och knappar. Jag önskar däremot att jag förbättrade min anpassning för mobilen, jag känner att jag fokuserade väldigt mycket på utseendet på datan och tänkte inte så mycket på mobilformat. Om jag hade redan från börjat tänkt på att anpassa hemsidan till mobilen, hade det varit inga problem men eftersom det dök upp i huvudet så sent, blir det svårt att anpassa då vissa koder behövs ändras och bytas ut för att det anpassas för både mobil och data. Jag försökte mitt bästa att anpassa den till mobilen men jobbet skulle gått mycket snabbare och enklare ifall jag tänkte på det innan.
